## üöÄ How to Use

1. **Navigate** to the desired program folder
2. **Open** the `.c` file
3. **Compile** using: `gcc filename.c -o output`
4. **Run** using: `./output` (Linux/Mac) or `output.exe` (Windows)

Example:
```bash
cd "Programs/Array"
gcc matrix_addition.c -o matrix_addition
./matrix_addition
```
